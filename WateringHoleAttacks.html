Watering Hole Attacks

  • A watering hole is a fake wireless access point, spoofed WiFi. This allows hackers to monitor, intercept, or steal their data streams.
  • This is not always a bogus WiFi, it can be an attractive website or social media platform. A watering hole is a way to harvest data or spread a malware infection to the maximum number of devices.

  • The Virtual Private Network (VPN) remains your safest option when using any wireless access.
